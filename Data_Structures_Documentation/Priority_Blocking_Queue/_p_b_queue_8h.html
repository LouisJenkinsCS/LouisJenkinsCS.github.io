Will attempt to add an element to the queue.

If it is bounded, it will block until it is no longer full. Note: It is recommended you use the Timed_Enqueue if your queue is bounded, due to the fact that it will block indefinitely until an element is removed.

int PBQueue_Timed_Enqueue ( PBQueue *  queue,
void *  item,
unsigned int  seconds 
)

Attempts to add an element to the queue and will return early if the time elapses.

This is the recommended way to deal with bounded queues, as it can timeout early, allowing you to avoid an infinite deadlock.

Parameters
queuePriority blocking queue to add an element to.
itemElement to be added to the priority blocking queue.
secondsMaximum time spent waiting to add an element.
Returns
1 if successful, 0 if timed out.
