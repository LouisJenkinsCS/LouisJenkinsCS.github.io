3 /*
4  * PBQueue is a minimal priority blocking queue, which will insert elements sorted
5  * in ascending order based on priority. The priority is determined by the comparator passed to it.
6  * It should be noted that this can be doubled as a normal blocking queue by having
7  * a comparator which just returns 0, like such.
8  *
9  * int comparator(void *item_one, void *item_two){
10  * return 0;
11  * }
12  *
13  * Since the priority blocking queue will detect whether it should be at the tail of the queue,
14  * doing so would insert it at the end with O(1) insertion. It should also be noted that
15  * if an item is not the lowest priority, the insertion is O(n).
16  *
17  * Alternatively, this can be used a regular blocking queue by calling the Timed_Enqueue
18  * and Timed_Dequeue with 0 seconds, causing it to timeout immediately if the queue is
19  * full/empty (respectively). A combination of the two could turn it into a normal
20  * queue with it's own quirks.
21  *
22  * Lastly, this Priority Blocking Queue can either be a bounded or unbounded queue
23  * depending on which constructor used to create it.
24  */
