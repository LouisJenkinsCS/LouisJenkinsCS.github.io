7 /*
8  * Miscallaneous Utilities for the C Programming Language, or Misc_Utils, is a collection,
9  * albeit a small one, of utilities that aren't big enough to deserve it's own package.
10  *
11  * Inside, it contains basic logging macros to log information to a file, a debugging
12  * macro to output information to stderr in a prettified format. It should also be
13  * noted that the logging functions also show the name of the file and the line where
14  * the macro is invoked, thanks to GCC magic.
15  *
16  * Inside also is a timer, which is extremely bare bones for now, and allows you
17  * to start, stop and obtain the total time in string format.
18  */
