1 /*
2  * @Author: Louis Jenkins
3  * @Version: 1.2
4  *
5  * String_Utils is a string library based off of Java's own String library, except
6  * the library works on char * over a string object (or struct). It is a lightweight,
7  * relatively speedy library which relies upon the libc and glibc.
8  *
9  * String_Utils also features passing flags with the '|' operator, for example,
10  * SU_IGNORE_CASE|SU_LAST does exactly as it says, ignore case for string comparison,
11  * and return the last string that matches the criteria. For default behavior, you
12  * pass SU_NONE flag instead.
13  *
14  * Some functions require passing the address of the string (&) for possible modification
15  * of the original string, as String_Utils never modifies the string passed, by default,
16  * so it is safe to use string literals unless passing SU_MODIFY flag.
17  */
